# 📜 [Server+DB] 익명 연봉/지출 비교 앱

> 🎯 **Server + DB (Supabase)** 를 조합해서 익명으로 연봉과 지출을 공유하고 비교할 수 있는 앱을 만드세요!

---

## 미션

1. 사용자가 **익명으로 연봉/지출 정보를 입력**할 수 있게 하세요 (직군, 연봉, 월 지출, 한마디 등)
2. 다른 사람의 데이터와 **평균/비교 통계**를 보여주세요 (전체 평균 연봉, 나의 위치 등)
3. **최신순 / 연봉순**으로 정렬해서 보여주세요
4. 모든 데이터는 **DB (Supabase)에 저장**하세요

---

## 핵심 구조

```
[익명 데이터 입력 (직군, 연봉, 지출)] → [Server] → DB에 저장 → 전체 목록 조회 → 평균/비교 통계 계산 → 화면 반영
```

- **DB 역할:** 익명 연봉/지출 데이터 저장 + 정렬/필터 조회
- **Server 역할:** 데이터 CRUD + 통계 계산 API

## 팁

> 💡 핵심은 **개인 데이터를 DB에 저장하고, 전체 평균과 비교**하는 흐름입니다. 내 연봉이 평균보다 높은지 낮은지 숫자로 보이는 순간 DB의 집계 쿼리가 체감됩니다.

- Supabase에 `salaries` 테이블을 만들고 `job_category`(직군), `salary`(연봉), `monthly_expense`(월 지출), `comment`(한마디), `created_at` 컬럼을 넣으세요
- 통계는 서버에서 `AVG(salary)`, `COUNT(*)` 등 SQL 집계 함수로 계산하면 깔끔합니다
- 연봉 입력 시 단위(만원)를 명확히 하면 데이터 비교가 훨씬 재미있어집니다
- 백분위(상위 몇 %)를 보여주면 창의성 포인트를 노릴 수 있습니다
