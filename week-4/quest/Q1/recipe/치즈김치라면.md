# 🍳 치즈김치라면

![치즈김치라면](치즈김치라면.png)

> 조리 시간: 10분 | 난이도: 자취생 | 1인분

## 재료 (냉장고에서 확인)

| 재료 | 수량 |
|------|------|
| 라면 | 1개 |
| 김치 | 100g (한 줌) |
| 계란 | 1개 |
| 슬라이스 치즈 | 1장 |
| 마늘 | 2쪽 |
| 대파 | 1/4대 |

## 조리 순서

1. 냄비에 물 550ml를 끓인다.
2. 마늘을 편으로 썰어 물이 끓기 전에 넣고 함께 끓인다.
3. 물이 끓으면 라면 스프와 김치를 넣는다.
4. 면을 넣고 2분 후 계란을 깨서 올린다 (반숙 원하면 뚜껑 덮기).
5. 불을 끄기 직전 슬라이스 치즈를 면 위에 올려 여열로 녹인다.
6. 대파를 송송 썰어 위에 뿌리고 완성.

## 팁

- 김치는 묵은지일수록 국물 맛이 깊어집니다.
- 치즈는 불 끈 후 올려야 타지 않고 쫀득하게 녹아요.
- 마늘을 미리 넣으면 국물이 훨씬 칼칼해집니다.
- 비엔나 소시지를 반 갈라 같이 넣으면 더 든든해요!
