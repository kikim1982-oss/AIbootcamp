# 📜 [Server+DB+AI] 냉장고 재료 기반 AI 레시피 제작앱

> 🎯 **Server + DB (Supabase) + AI** 를 조합해서 냉장고 재료 기반으로 AI가 레시피를 자동 생성해주는 앱을 만드세요!

---

## 미션

1. 이전 퀘스트에서 만든 **재료 관리 앱을 기반으로** 시작하세요
2. 저장된 재료를 기반으로 **AI가 레시피를 자동 생성**해주는 기능을 추가하세요
3. AI가 생성한 레시피를 **DB에 저장**할 수 있게 하세요
4. 저장된 레시피를 **목록으로 조회**할 수 있게 하세요

---

## 핵심 구조

```
[DB에서 재료 조회] → [Server: AI API 호출] → [레시피 자동 생성] → [DB에 저장] → [레시피 목록에 표시]
```

- **DB 역할:** 재료 저장 + AI가 생성한 레시피 저장/관리
- **Server 역할:** DB에서 재료 조회 → AI API 호출 → 생성된 레시피 DB에 저장
- **AI 역할:** 저장된 재료 목록을 받아 만들 수 있는 레시피 추천/생성

- AI가 만든 레시피가 마음에 들면 저장, 아니면 다시 생성 — **사용자가 선택하는 흐름**을 만들어보세요
