# 📜 [Server+DB] 익명 고민/칭찬 게시판

> 🎯 **Server + DB (Supabase)** 를 조합해서 익명 고민/칭찬 게시판을 만드세요!

---

## 미션

1. 사용자가 **익명으로 글을 작성**할 수 있게 하세요 (고민, 칭찬, 응원 등 카테고리 선택)
2. 다른 사람이 글에 **공감 버튼**을 누를 수 있게 하세요 (공감 수 +1 업데이트)
3. **최신순 / 공감순**으로 정렬해서 보여주세요
4. 모든 데이터는 **DB (Supabase)에 저장**하세요

---

## 핵심 구조

```
[글 작성 (카테고리, 내용)] → [Server] → DB에 저장 → 게시글 목록 조회 → 공감 버튼 클릭 → DB UPDATE (공감 수 +1) → 화면 반영
```

- **DB 역할:** 게시글 저장 + 공감 수 관리 + 정렬/필터 조회
- **Server 역할:** 게시글 CRUD + 공감 업데이트 API

## 팁

> 💡 공감 버튼의 핵심은 DB의 **UPDATE** 입니다. 버튼 한 번 누를 때마다 숫자가 올라가는 걸 보면 "DB에 저장된 데이터를 수정한다"는 개념이 체감됩니다.

- Supabase에 `posts` 테이블을 만들고 `category`, `content`, `likes` (공감 수), `created_at` 컬럼을 넣으세요
- 공감 버튼 클릭 시 `likes = likes + 1` 로 UPDATE하면 됩니다
- 수업 시간에 서로의 글에 공감을 눌러보면 DB의 읽기/쓰기가 동시에 일어나는 경험을 할 수 있습니다
