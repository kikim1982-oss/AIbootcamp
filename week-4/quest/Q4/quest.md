# 📜 [Server+DB] 실시간 밸런스 게임 앱

> 🎯 **Server + DB (Supabase)** 를 조합해서 실시간 밸런스 게임 앱을 만드세요!

---

## 미션

1. 밸런스 게임 **질문을 등록**할 수 있게 하세요 (예: "월급 500 + 주7일 출근 vs 월급 300 + 주4일 출근")
2. 사용자가 둘 중 하나에 **투표**하면 실시간으로 퍼센티지가 변하게 하세요
3. **투표 결과와 총 참여자 수**를 보여주세요
4. 모든 데이터는 **DB (Supabase)에 저장**하세요

---

## 핵심 구조

```
[질문 등록 (A vs B)] → [투표 선택] → [Server] → DB에 저장 → 실시간 투표율 계산 → 퍼센티지 바 업데이트
```

- **DB 역할:** 질문 저장 + 투표 결과 기록 + 실시간 집계
- **Server 역할:** 질문/투표 CRUD + 투표율 계산 API
