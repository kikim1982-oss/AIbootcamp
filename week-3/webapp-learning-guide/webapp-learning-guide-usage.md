# webapp-learning-guide 에이전트 사용법

## 이 에이전트가 뭔가요?

초보 웹 개발자를 위한 **학습 멘토 에이전트**입니다.
프로젝트 폴더의 실제 코드를 읽고, 한국어로 친절하게 설명해줍니다.

---

## 언제 사용하나요?

| 상황 | 예시 질문 |
|------|-----------|
| 특정 파일이 뭐 하는지 모를 때 | `"server.js 파일이 뭐 하는 건지 모르겠어"` |
| 개념이 이해가 안 될 때 | `"API가 뭐야?"`, `"라우팅이 뭐야?"` |
| 이 프로젝트에서 어떻게 동작하는지 알고 싶을 때 | `"이 프로젝트에서 React 어떻게 쓰고 있어?"` |
| 이전에 배운 내용 다시 찾고 싶을 때 | `"저번에 Express 미들웨어 배웠는데 다시 알려줘"` |

---

## 사용 방법

Claude Code 채팅창에 그냥 질문하면 됩니다.
Claude가 자동으로 이 에이전트를 실행합니다.

### 예시

```
server.js 파일이 뭐 하는 건지 모르겠어
```

```
이 프로젝트에서 fetch가 어떻게 쓰이고 있어?
```

```
REST API가 뭐야? 이 프로젝트에서 예시 보여줘
```

```
저번에 Express에 대해 배웠는데 다시 찾아줘
```

---

## 에이전트의 특징

- **실제 코드 기반 설명** — 추상적인 설명 대신 이 프로젝트의 실제 파일을 열어서 설명
- **한국어 응답** — 기본적으로 한국어로 답변
- **초보자 친화적** — 전문 용어는 비유로 설명 (예: "서버는 식당 주방 같은 거예요")
- **학습 기록 저장** — 배운 내용을 기억해뒀다가 나중에 다시 물어볼 수 있음
