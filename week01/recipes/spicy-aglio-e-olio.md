# 매콤 알리오올리오 (Spicy Aglio e Olio)

> *"불의 온도를 다스리는 자만이 마늘의 진정한 향을 끌어낼 수 있다."*

나폴리 빈민가의 셰프들이 마늘과 올리브유, 그리고 건고추만으로 창조해낸 이 파스타는 '가난의 미학'이라 불린다. 약불 위에서 천천히 황금빛으로 익어가는 마늘 슬라이스, 그 위로 페페론치노의 매운 향이 피어오르는 순간이야말로 이 요리의 절정이다. 여기에 한국의 고추기름과 청양고추가 더해져, 동서양의 매운맛이 하나의 접시 위에서 조우하는 특별한 한 끼가 탄생한다.

---

**조리 시간** 15분 · **서빙** 1인분 · **난이도** Easy

---

## Ingredients

| | |
|:------|:------|
| 스파게티 면 — 100g | 마늘 — 4~5쪽 |
| 올리브유 — 3~4큰술 | 소금 — 약간 (면 삶는 물용) |
| 후추 — 약간 | 페페론치노(건고추) — 2~3개 |
| 고추기름 — 1큰술 *(고춧가루 1작은술로 대체 가능)* | 청양고추 — 1개 *(생략 가능)* |
| 파슬리 — 약간 *(생략 가능)* | |

---

## Method

![cooking](./thumbnails/spicy-aglio-e-olio-cooking.png)

| Classic | Light |
|:------|:------|
| **01** 냄비에 물을 넉넉히 붓고 강불로 가열한다. 물이 끓으면 소금 1큰술을 투입하여 간을 맞춘다. | **01** 동일하게 진행한다. 소금은 1/2큰술로 절제하여 나트륨을 조절한다. |
| **02** 마늘을 얇게 슬라이스한다. 칼 옆면으로 가볍게 으깬 뒤 썰면 보다 수월하게 작업할 수 있다. | **02** 동일하게 진행한다. 마늘은 열량이 미미하므로 풍미를 위해 충분히 사용한다. |
| **03** 면을 포장지 표기 시간보다 1분 일찍 건져내어 알덴테 식감을 살린다. 면수 한 국자를 반드시 보관한다. | **03** 스파게티 대신 곤약면 또는 통밀면(80g)으로 대체하여 열량을 현저히 낮춘다. |
| **04** 팬에 올리브유와 고추기름을 두르고 약불에서 마늘, 페페론치노, 청양고추를 천천히 볶는다. 매운 향이 피어오르면 적절한 시점이다. | **04** 올리브유 1작은술만 사용하고 고추기름은 제외한다. 눌어붙음 방지를 위해 논스틱 팬을 활용한다. |
| **05** 삶은 면을 팬으로 옮기고 보관해둔 면수를 조금씩 더하며 소스와 고루 결합시킨다. | **05** 동일하게 진행한다. 면수를 넉넉히 활용하여 기름 감소분을 보완한다. |
| **06** 후추를 갈아 올리고 파슬리를 散布하여 마무리한다. | **06** 후추와 파슬리로 마무리한다. |

## Chef's Note

| Classic Tips | Light Tips |
|:------|:------|
| **면수는 소스의 핵심이다** — 면수 약 100ml를 확보해두는 것이 에멀전 형성의 관건이다. | 곤약면 사용 시 별도의 면수가 없으므로, 물 100ml에 소금 약간을 녹여 대용한다. |
| **화력은 반드시 약불로** — 강불에서는 마늘이 순식간에 탄화되어 쓴맛을 유발한다. | 올리브유 1작은술과 논스틱 팬의 조합만으로 충분한 결과를 얻을 수 있다. |
| **설거지 최소화** — 냄비 하나에서 면을 삶은 뒤 그대로 소스를 완성하면 세척 부담을 크게 줄일 수 있다. | 세척 효율화 전략은 동일하게 적용된다. 냄비 일원화를 권장한다. |
| **매운맛의 강도 조절** — 페페론치노를 분쇄하여 씨까지 함께 투입하면 더욱 강렬한 자극을 연출할 수 있다. | 고추기름을 배제하고 고춧가루만 사용해도 매운맛의 본질은 유지되며 열량은 절감된다. |
| **올리브유 대체** — 부재 시 일반 식용유로 대체 가능하다. | 통밀면 또는 곤약면으로 교체하면 열량을 50% 이상 절감할 수 있다. |
| **면의 선택** — 소면이나 다른 파스타 면으로도 동일한 방식으로 응용할 수 있다. | 애호박을 줄리엔느 커팅하여 면 대용으로 활용하면 초저열량 버전이 완성된다. |
