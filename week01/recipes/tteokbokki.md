# 떡볶이

> *"불의 온도와 시간이 평범한 떡을 비범한 한 끼로 완성한다."*

1950년대 서울 종로의 작은 분식집에서 시작된 떡볶이는, 고추장이라는 단 하나의 양념으로 대한민국 길거리 음식의 역사를 다시 썼다. 쫄깃한 떡이 붉은 소스를 머금고 윤기를 띠는 그 순간, 소박한 재료는 중독성 있는 한 접시로 거듭난다. 누군가에게는 학창 시절의 추억이고, 누군가에게는 하루의 고단함을 달래주는 따뜻한 위로 — 그것이 떡볶이의 본질이다.

---

**조리 시간** 15분 · **서빙** 1인분 · **난이도** Easy

## Ingredients

| | |

| 간장 — 1 작은술 | 설탕 — 1 큰술 |
| 대파 — 약간 (생략 가능) | |

## Method

![cooking](./thumbnails/tteokbokki-cooking.png)

| Classic | Light |
|:------|:------|
| **01** 떡이 단단할 경우, 찬물에 5분간 침지하여 질감을 부드럽게 회복시킨다. | **01** 떡볶이 떡 대신 곤약떡(150g)으로 대체. 찬물로 헹궈 잡내를 제거한다. |
| **02** 어묵을 일정한 크기로 절단하여 균일한 조리가 되도록 준비한다. | **02** 어묵 대신 두부 또는 삶은 달걀을 활용하여 단백질을 보강한다. |
| **03** 냄비에 물, 고추장, 간장, 설탕을 투입하고 소스가 균일하게 혼합되도록 충분히 저어준다. | **03** 설탕을 알룰로스로 대체하고, 고추장은 1큰술로 조정하여 당도와 자극을 절제한다. |
| **04** 중불에서 소스가 끓어오르면 떡과 어묵을 투입한다. | **04** 동일하게 진행하되, 양배추 또는 팽이버섯을 함께 추가하여 포만감을 높인다. |
| **05** 중약불로 조정 후 7~8분간 지속적으로 저어가며 소스를 졸인다. | **05** 곤약떡은 열전달이 빠르므로 5분의 조리로 충분하다. |
| **06** 소스가 충분히 농후해지고 떡의 질감이 유연해지면 완성. 대파를 올려 마무리한다. | **06** 동일하게 마무리. 대파를 올려 시각적 완성도를 더한다. |

## Chef's Note

| Classic Tips | Light Tips |
|:------|:------|
| 냄비 하나로 완성되는 원팟 조리로 설거지를 최소화한다. | 곤약떡은 열량이 극히 낮아 부담 없이 즐길 수 있다. |
| 눌어붙음을 방지하기 위해 조리 중 지속적으로 저어주는 것이 핵심이다. | 양배추, 팽이버섯 등 채소를 더하면 볼륨은 늘고 칼로리는 낮아진다. |
| 고추장의 투입량으로 매운 강도를 섬세하게 조절할 수 있다. | 알룰로스가 없는 경우 스테비아 또는 설탕 1/2큰술로 충분히 대체 가능하다. |
| 어묵 외에도 삶은 달걀, 햄, 두부 등 다양한 단백질 재료로 변주가 가능하다. | 남은 소스에 곤약밥을 더하면 저칼로리 마무리 한 그릇이 완성된다. |
| 남은 소스에 밥을 비벼 내면 훌륭한 마무리 요리가 된다. | |
