# 📜 [Auth+MCP+DB+App] 나만의 AI 대시보드

> 🎯 5주차에 배운 **Auth + MCP + DB + AI**를 하나의 앱에 종합하는 **보스 퀘스트**! 로그인하면 내 데이터가 한 화면에 모이고, AI가 브리핑해주는 **개인 대시보드**를 만들어보세요.

## 전제 조건

> ⚠️ 이 퀘스트는 **도전 퀘스트**입니다. 5주차 다른 퀘스트를 먼저 완수한 수강생에게 추천합니다.
> 모든 기능을 완벽하게 구현하지 않아도 됩니다. 대시보드에 **2~3개 위젯만 동작해도 인정**!

## 미션

### Part 1: 로그인/회원가입 구현 (Auth)

대시보드에 접근하려면 로그인이 필요하도록 만드세요.

### Part 2: 데이터 소스 연결

아래 중 2개 이상을 대시보드에 연결하세요:

- **Notion MCP:** 내 할일/메모 데이터 연동
- **DB (Supabase):** 개인 데이터 저장 (지출, 습관, 메모 등)
- **외부 API:** 날씨, 뉴스 등

### Part 3: AI 브리핑 생성

AI가 연결된 데이터를 종합해서 **"오늘의 브리핑"**을 생성하도록 만드세요.

### Part 4: 대시보드 UI & 배포

하나의 웹앱 대시보드에 모든 것을 표시하고, Vercel에 배포하세요.

**대시보드 구성 예시:**

```
┌─────────────────────────────────────────┐
│  좋은 아침, OO님!          [로그아웃]     │
├──────────────┬──────────────────────────┤
│  오늘의 할일   │  AI 브리핑               │
│  (Notion 연동) │  "오늘 회의 2개 있고,     │
│  □ 기획서 작성  │   이번 주 지출이 예산의    │
│  □ 팀 미팅     │   80%에 도달했어요.       │
│  ☑ 코드 리뷰   │   점심은 가볍게 드세요!"   │
├──────────────┼──────────────────────────┤
│  이번 주 지출   │  최근 활동               │
│  (DB 연동)     │  (DB 연동)               │
│  식비: 45,000  │  어제: 운동 30분          │
│  교통: 12,000  │  그제: 독서 1시간         │
└──────────────┴──────────────────────────┘
```

## 핵심 구조

```
[로그인] → [Auth 확인] → [Notion MCP + DB + API] → [AI가 데이터 종합] → [대시보드에 표시]
```

- **Auth:** 내 대시보드에 나만 접근
- **MCP:** 외부 서비스 데이터 연동
- **DB:** 개인 데이터 저장/조회
- **AI:** 모든 데이터를 종합해서 브리핑 생성
