# 📜 [Server+DB] 가계부 앱

> 🎯 **Server + DB (Supabase)**를 조합해서 나만의 가계부 앱을 만드세요!

---

## 미션

1. 사용자가 **수입/지출 내역을 등록**할 수 있게 하세요 (날짜, 금액, 카테고리, 메모)
2. 등록된 내역을 **목록으로 조회**할 수 있게 하세요
3. **카테고리별 합계**를 보여주세요 (식비, 교통, 주거, 구독료, 경조사 등)
4. 모든 데이터는 **DB (Supabase)에 저장**하세요

---

## 핵심 구조

```
[사용자 입력 (금액, 카테고리, 메모)] → [Server] → DB에 수입/지출 저장 → 내역 조회 & 카테고리별 합계 계산 → 결과 응답
```

- **DB 역할:** 수입/지출 내역 저장소 + 카테고리별 통계 조회
- **Server 역할:** CRUD API 제공 (등록, 조회, 수정, 삭제)

## 팁

> 💡 가계부의 핵심은 **카테고리 분류**입니다! 수입/지출을 구분하고, 지출은 세부 카테고리로 나누면 내 돈이 어디로 새는지 한눈에 보입니다.

- Supabase에 테이블을 만들 때 `type` (수입/지출), `category` (식비, 교통, 주거, 구독료, 경조사 등), `amount`, `memo`, `date` 컬럼을 넣으세요
- 카테고리별 합계는 SQL의 `GROUP BY`를 활용하면 간단합니다
- 여유가 되면 월별 지출 차트나 예산 대비 사용량 표시에 도전해보세요
