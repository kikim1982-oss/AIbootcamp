## Step 1. 관심 주제 확정

**가이드라인:**
- 매주/매일 반복해서 보고 싶은 사이트가 있는 주제일수록 좋음
- 데이터가 표 형태로 추출되기 쉬운 주제가 정리에 유리
- "Quest 1처럼 학습+업무 데이터 → Quest 2는 외부 정보" 식으로 차별화

**후보 주제 (예시 — 본인 상황에 맞게 교체):**

| # | 주제 | 사이트 후보 | 활용 시나리오 |
| --- | --- | --- | --- |
| A | AI 관련 채용공고 | 원티드, 사람인, 잡코리아 | 매주 신규 공고 모니터링 |
| B | 동네 맛집 리뷰 | 네이버지도, 망고플레이트, 카카오맵 | 평점·리뷰 기반 정렬 |
| C | IT/AI 뉴스 헤드라인 | 테크크런치, 더버지, 자취IT뉴스, 긱뉴스 | 매일 아침 5분 브리핑 |
| D | 중고 상품 가격 비교 | 당근마켓, 번개장터, 헬로마켓 | 특정 키워드 시세 추적 |
| E | 부동산 매물 | 직방, 다방, 네이버부동산 | 조건 매물 알림 |
| F | 항공권 가격 | 스카이스캐너, 네이버항공권, 인터파크 | 노선별 최저가 추적 |
| G | 주식·암호화폐 시세 + 뉴스 | 네이버증권, 야후파이낸스, 코인마켓캡 | 매일 시황 요약 |

## Step 2. 탐색할 사이트 3곳+ 후보 선정

선택한 주제에 따라 사이트 3~4곳 확정.

**좋은 사이트 조건:**
- 로그인 안 해도 콘텐츠 보임 (공개 페이지)
- 검색·필터 URL이 명확 (예: `?keyword=ai`)
- 결과가 리스트 형태 (카드/테이블 → 추출 쉬움)

**산출물:**
```markdown
## 탐색 대상 사이트
1. 사이트명 — URL — 추출할 항목 (제목, 가격, 평점 등)
2. ...
3. ...
```

## Step 3. 수집할 데이터 항목 설계

각 사이트에서 동일한 스키마로 추출하면 통합 정리 쉬움.

**예시 (채용공고 주제):**
| 필드 | 타입 | 비고 |
| --- | --- | --- |
| 회사명 | text | |
| 직무 | text | |
| 경력 요건 | text | "신입/3년+" |
| 연봉 | text | 표시 안 된 경우 "협의" |
| 위치 | text | |
| 마감일 | date | |
| 원본 링크 | url | |
| 출처 사이트 | text | "원티드"/"사람인" 등 |

→ Step 4부터 이 스키마에 맞춰 데이터 추출

## Step 4~6. 사이트별 자동 탐색 (각 사이트 동일 흐름)

각 사이트마다:

1. **명령 입력** — Claude에게 자연어로 요청
   - 예: "원티드에서 AI 엔지니어 채용공고 상위 10건 추출해서 표로 정리해줘"
2. **탐색 진행** — Chrome MCP가 브라우저 열고 페이지 이동/스크롤/추출 자동 수행
3. **스크린샷** — 자동 탐색 진행 중인 브라우저 화면 캡처 (Quest 제출 필수)
4. **결과 검토** — 누락·오류 있으면 후속 질문으로 보완 (활용 5pt 확보)

**스크린샷 저장 규칙:**
- 폴더: `week-5/Q2관심분야자동리서치/screenshots/`
- 파일명: `site1-search.png`, `site1-detail.png`, `site2-search.png` …
- 최소 3장 (사이트별 1장씩) — Quest 제출 필수

## Step 7. 결과 통합 → `research-result.md`

수집한 데이터를 통합해서 1개의 md 파일로 정리.

**파일 구조 예시:**
```markdown
# 🔍 [주제명] 자동 리서치 결과

> 수집 일시: YYYY-MM-DD HH:MM
> 탐색 사이트: 3곳
> 총 결과 수: N건

## 📊 통합 결과 표 (모든 사이트 공통 포맷)
| 출처 | 항목명 | 핵심정보 | 링크 |
| --- | --- | --- | --- |
| ... |

## 🏷️ 사이트별 상세
### 1. 사이트A (N건)
- ...
### 2. 사이트B (N건)
### 3. 사이트C (N건)

## 💡 인사이트
- 사이트별 차이점
- 가장 매력적인 항목 TOP 3
- 공통 트렌드

## 🤖 자동화 아이디어
- "매주 월요일 아침 자동 실행하면 좋을 것"
```

## Step 8. (보너스) Notion 업로드
